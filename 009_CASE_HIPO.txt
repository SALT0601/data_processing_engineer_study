요구사항 분석 CASE와 HIPO

CASE(자동화 도구)
- 요구사항을 자동으로 분석, 분석 명세서를 기술하도록 개발된 도구
- 대표
=> SADT : 정의, 분석, 설계를 위한 도구로 SoftTech사에서 개발. 블록 다이어그램 채택
     SREM : RSL/REVS : TRW사가 요구사항을 명확히 기술하도록 개발. RSL(요구사항 기술언어), REVS(요구사항 분석기) 사용
     PSL/PSA : 미시간 대학에서 개발. PSL(언어), PSA(분석기) 사용
     TAGS : 시스템 공학 방법 응용에 대한 자동 접근 방법. 개발 주기 전 과정에 이용하는 통합 자동화 도구

HIPO(Hierarchy Input Process Output)
- 분석,설계,문서화에 사용. 시스템 실행과정인 입력,처리,출력 기능 표현
- 하향식 문서화 도구
- 기능과 자료의 의존관계 동시 표현 가능
- 기호, 도표 사용해 이해하기 쉬움
- HIPO Chart : 기능을 여러 모듈로 분할해 계층 구조로 표현한 것 
- HIPO Chart 종류
=> 가시적 도표, 총체적 도표, 세부적 도표
