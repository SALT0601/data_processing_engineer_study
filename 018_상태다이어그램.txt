상태(State) 다이어그램

상태 다이어그램
- 객체들 사이에 발생하는 이벤트에 의한 객체들의 상태 변화 그림으로 표현 => 상호 작용에 따른 객체 상태 변화 표현
- 상태 : 객체가 갖는 속성 값의 변화
- 어떤 이벤트에 의해 상태 변환 과정이 진행되는지 확인
- 상태 변환 이벤트를 확인할 필요가 있는 객체만 그림

구성요소
- 상태 - 둥근 모서리 사각형 - 객체 상태
- 시작 상태 - ● - 상태 시작
- 종료 상태 - 두겹 원형 - 상태 종료
- 상태 전환 - 화살표 - 상태 사이의 흐름, 변화
- 이벤트 - 글씨 - 상태에 변화를 주는 현상 (조건, 외부 신호, 시간의 흐름 등)
- 프레임 - 사각형(p.85 참고) - 상태 다이어그램의 범위
