SW 관련 신기술
- 인공지능(AI) : 컴퓨터 스스로 추론해 작업 수행(패턴 인식, 전문가 시스템 등)
- 뉴럴링크 : 테슬라 일론 머스크가 뇌와 컴퓨터 결합 기술 개발 위해 설립한 회사
- 딥러닝 : 인간의 두뇌를 모델로 만들어진 인공 신경망 기반으로 하는 기계 학습 기술
- 전문가 시스템 : 의료 진당 등 특정 분야 전문가가 수행하는 고도의 업무 지원
- 증강현실(AR) = 혼합현실 : 실제 촬영한 화면에 가사의 정부 부가해 보여줌
- 블록체인 : P2P 네트웨크 이용해 온라인 금융 거래 정보를 온라인 네트워크 참여자(Peer)의 디지털 장비에 분산 저장
- 분산 원장 기술(DLT) : 중앙 관리자나 데이터 중앙 데이터 저장소 존재하지 않고 P2P 망 내 참여자들에게 분산 저장되어 거래 발생때마다 지속적으로 갱신
- 해시 : 임의의 길이의 입력 데이터나 메시지를 고정된 길이의 값이나 키로 변환
- 양자 암호키 분배(QKD) : 양자 통신을 위해 비밀키 분배해 관리
- 프라이버시 강화 기술(PET) : 개인정보 침해 위험을 관리하기 위한 핵심 기술, 암호화 익명화 등
- 공통 평가 기준(CC) : 정보 보호 제품 평가 기준
- 개인정보 영향평가 제도(PIA) : 개인 정보 활용하는 정보 시스템 도입 시 사생활에 미칠 영향에 대해 미리 조사, 분석, 평가하는 제도
- 그레이웨어 : 제공하는 입장과 달리 사용자 입장에서는 유용할 수도 있고 악의적일 수 있는 애드웨어, 트랙웨어 등 악성 코드나 악성 공유웨어
- 매시업 : 웹에서 제공하는 정보 및 서비스 이용해 새로운 소프트웨어나 서비스 등 만드는 기술
- 리치 인터넷 애플리케이션(RIA) : 플래시 애니메이션 기술과 웹 서버 애플리케이션 기술 통합해 역동적이고 인터랙티브한 웹페이지 제공
- 시맨틱 웹 : 컴퓨터가 사람을 대신해 정보 읽고 이해하고 가공해 새로운 정보 만들어 낼 수 있도록 이해하기 쉬운 의미를 가진 차세대 지능형 웹
- 증발품 : 판매 계획 또는 배포 계획은 발표되었으나 실제로 고객에게 판매되거나 배포되지 않고 있는 소프트웨어
- 오픈 그리드 서비스 아키텍처(OGSA) : 웹 서비스를 그리드 상에서 제공하기 위한 개방형 표준
- 서비스 지향 아키텍처(SOA) : 기업의 소프트웨어 인프라인 정보시스템을 서비스 단위나 컴포넌트 중심으로 구축하는 정보기술 아키텍처
	     	 	 : SOA 기반 애플리케이션 구성 계층 - 표현 - 업무 프로세스 - 서비스 중간 - 애플리케이션 - 데이터 저장
- 서비스형 소프트웨어(SaaS) : 사용자가 필요로하는 서비스만 이용할 수 있도록한 소프트웨어
- 소프트웨어 에스크로: 개발자의 지식재산권 보호 등 위해 소스 프로그램과 기술 정보 등을 제3의 기관에 보관
- 복잡 이벤트 처리(CEP) : 의미가 있는 것만을 추출할 수 있도록 사건 발생 조건을 정의하는 데이터 처리
- 디지털 트윈 : 현실속 사물을 소프트웨어로 가상화한 모델, 실제 물리적 자신을 가상화해서 특성에 대한 정확한 정보 얻을 수 있음(자동차, 항공 등 주목)
