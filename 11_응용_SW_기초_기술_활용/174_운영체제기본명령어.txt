운영체제 기본 명령어

Windows 기본 명령어
- DIR : 현재 디렉터리의 파일 목록 표시
- COPY : 파일 복사
- DEL : 파일 삭제
- TYPE : 파일의 내용 표시
- REN : 파일의 이름 변경
- MD : 디렉터리 생성
- CD : 동일한 드라이브에서 디렉터리 위치 변경
- CLS : 화면의 내용 지움
- ATTRIB : 파일의 속성 변경(속성 종류 - R(읽기 전용), A(저장,백업), S(시스템 파일), H(숨김 파일) -> 지정시 속성앞에 + 해제시 앞에 - 
- FIND : 파일에서 문자열 찾음
- CHKDSK : 디스크 상태 점검
- FORMAT : 디스크 표면을 트랙과 섹터로 나누어 초기화
- MOVE : 파일 이동

UNIX/LINUX 기본 명령어
- cat : 파일 내용 화면에 표시
- cd : 디렉터리 위치 변경
- chmod : 파일 보호 모드 설정해 파일의 사용 허가 지정함
  -> 사용자(u(user), g(group), o(other), a(all)) / 설정기호(+(권한추가), -(권한삭제), =(권한부여)) / 권한(r(read), w(write), x(실행))
- chown : 파일 소유자와 그룹 변경
- cp : 파일 복사
- rm : 파일 삭제
- find : 파일 찾음
- fsck : 파일 시스템 검사하고 보수
- kill : PID(프로세스 고유 번호) 이용해 프로세스 종료
- fork : 새로운 프로세스 생성
- killall : 프로세스 이름 이용해 프로세스 종료
- ls : 현재 디렉터리의 파일 목록 표시
- mkdir : 디렉터리 생성
- rmdir : 디렉터리 삭제
- mv : 파일 이동
- ps : 현재 실행중인 프로세스 표시
- pwd : 현재 작업중인 디렉터리 경로 화면에 표시
- top : 시스템의 프로세스와 메모리 사용 현황 표시
- who : 현재 시스템에 접속해 있는 사용자 표시

- UNIX에서 파일의 권한 10자리로 표현(1 : 디렉터리(D) 또는 파일(-)/ 2~4 : 소유자 권한/ 5~7 : 그룹권한/ 8~10 : 기타 사용자 권한)
- 각 자리는 r, w, x, -(권한없음)으로 표시
-> ex. r-x : 읽기, 실행만 가능
- 권한을 변경하는 chmod 명령어는 위의 권한 표현 방식을 8진수로 변경해 사용
- 변경 방법은 파일 구분을 제외한 각 권한을 권한 있음(1), 권한없음(0)으로 바꾼뒤 8진수로 변환해 chmod 명령어의 매개 변수로 사용
