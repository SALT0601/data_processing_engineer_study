프로젝트 일정 계획
- 프로젝트의 프로세스를 이루는 소작업 파악하고 노력을 소작업에 분배해 소작업의 순서와 일정 정함
- 사용되는 기능 : WBS(프로젝트를 작은 관리 단위로 분할하여 계층적으로 기술한 업무 구조)
                      PERT/CPM(지연방지하고 께획대로 진행되도록 일정 계획, 대단위 계획의 조직적 추진 비용을 적게 사용하며 최단시간 내 계획 완성 위함)
                      간트 차트 등

PERT(Program Evaluation and Review Technique, 프로그램 평가 및 검토 기술)
- 전체 작업의 상호관계를 표시하는 네트워크
- 낙관적인 경우, 가능성 있는 경우, 비관적인 경우 로 단계 나누어 종료시기 결정
- 소요 기간 예측이 어려운 경우 사용
- 노드와 간선으로 구성, 원 노드는 작업, 간선에는 낙관치, 기대치, 비관치 표시
- 결정경로, 작업에 대한 경계시간, 작업 간 상호 관련성(ex. C는 B 작업 진행 후 수행) 등을 알 수 있음
- 작업 예측치 계산 공식
=> 작업 예측치 = 비관치+4*기대치+낙관치 / 6   평방 편차 = [(비관치-낙관치) / 6]^2

CPM(Critical Path Method, 임계 경로 기법) -> 임계 경로는 시간이 제일 오래걸리는 최장 경로
- 필요 작업 나열하고 작업에 필요한 소요 기간을 예측하는데 사용
- 노드와 간선으로 구성, 노드는 작업, 간선은 작업 사이의 전후 의존 관계 
- 원형 노드는 각각의 작업, 작업 이름과 소요 기간 표시
- 박스 노드는 이정표, 이정표 이름과 예상 완료 시간 표시
- 간선을 나타내는 화살표 흐름에 따라 작업 진행, 전 작업이 완료되어야 다음 작업 진행 가능

간트 차트 = 시간선 차트
- 프로젝트의 작업들이 언제 시작하고 종료되는지 작업 일정을 막대 도표로 표시
- 중간 목표 미달성 시 이유와 기간 예측 가능함
- 문제점이나 예산 초과 지출 등 관리, 자원 배치와 인원 계획에 유용
- 이정표, 작업 일정, 작업 기간, 산출물로 구성
- 수평 막대 길이는 각 작업의 기간
