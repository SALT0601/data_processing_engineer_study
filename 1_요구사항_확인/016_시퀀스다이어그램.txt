시퀀스(Sequence) 다이어그램

동적 모델링
- 내부 구성 요소들의 상태 변화 과정과 과정에서 발생하는 상호 작용을 그림으로 표현
  (기능 - 기능 표현 / 정적 - 구성 요소 표현 / 동적 - 상태 변화 표현)
- 동작이라는 관점(View)에서 표현
- 오퍼레이션(메시지 호출)을 통한 상호 작용에 초점
- 종류
=> 시퀀스 다이어그램, 커뮤니케이션 다이어그램, 상태 다이어그램

시퀀스 다이어그램
- 시스템이나 객체들이 메시지를 주고받으며 상호 작용하는 과정 그림으로 표현 
- 상호작용 과정에서 주고 받는 메시지 표현
- 수행 기간 확인 가능
- 객체들을 기본 단위로 하여 상호작용 표현 => 클래스가 수행할 수 있는 오퍼레이션 표현
- 메시지에 의한 상호 작용 표현 - 시퀀스, 커뮤니케이션 / 동기에 의한 상태 변화 표현 - 상태

구성요소
- 액터 - 사람 모양 - 외부요소(사람, 외부 시스템)
- 객체 - 사각형 - 메시지 주고받는 주체
- 생명선 - 점선 - 객체가 메모리에 존재하는 기간, 객체 아래에 표시, 객체 소멸 기간까지 존재
- 실행 상자(활성 상자) - 생명선에 겹친 직사각형 - 메시지 주고 받으며 구동되고 있음을 표현
- 메시지 - 화살표 - 주고 받는 메시지
           - 점선 화살표 - 반환 메시지
=> 묵시적으로 위쪽 메시지가 먼저 전달되는 메시지
- 객체 소멸 - 엑스표시 - 객체가 메모리에 존재하지 않음
- 프레임 - 사각형(p.79 참고) - 다이어그램 전체나 일부 묶어 표현
