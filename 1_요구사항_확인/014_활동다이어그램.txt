활동(Activity) 다이어그램

활동 다이어그램
- 사용자의 관점에서 기능을 처리 흐름에 따라 순서대로 표현
- 하나의 유스케이스 안에서 혹은 유스케이스 사이에서 복잡한 처리의 흐름 명확히 표현
- 자료 흐름도(순서도)와 유사

구성요소
- 액션/액티비티 
=> 액션 - 둥근 모서리 사각형 - 분해할 수 없는 단일 작업
     액티비티 - 둥근 모서리 사각형 - 몇 개의 색션으로 분리 가능
- 시작 노드 - ● - 액션, 액티비티 시작
- 종료 노드 - 두겹 채워진 원형 - 모든 흐름 종료 
- 조건(판단) 노드 - 마름모와 화살표 - 조건에 따라 제어 흐름 분리, 들어오는 것 하나이고 나가는 것 여러 개
- 병합 노드 - 마름모와 화살표 - 경로의 흐름이 합쳐짐, 들어오는 것 여러개이고 나가는 것 하나
- 포크(Fork) 노드 - 직선과 화살표 - 액티비티의 흐름 분리 수행, 들어오는 것 하나이고 나가는 것 여러 개
- 조인(Join) 노드 - 직선과 화살표 - 분리 수행한 액티비티 흐름 합쳐짐, 들어오는 것 여러 개이고 나가는 것 하나
- 스윔레인(Swim Lane) - 가로 또는 세로 실선 - 액티비티 수행 주체를 구분, 시스템별 처리 흐름 진행되는 부분 구분 위함. 마치 수영장 레인처럼
