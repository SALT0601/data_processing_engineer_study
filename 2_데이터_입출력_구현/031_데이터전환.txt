데이터 전환 = 데이터 이행 = 데이터 이관
- 현재 운영 중인 시스템의 데이터를 추출(Extraction)하고 이를 변환(Transformation)한 후, 개발할 시스템에서
  사용할 수 있도록 적재(Loading)하는 과정 -> ETL 과정

데이터 전환 계획서
- 데이터 적환 작업에 필요한 모든 계획을 기록하는 문서 -> 데이터 전환을 성공적으로 수행하기 위함
- 주요항목
=> 데이터 전환 개요, 데이터 전환 대상 및 범위, 전환 환경 구성, 전환 조직 및 역할, 전환 일정,
     전환 방안, 정비 방안, 비상 계획, 복구 대책
