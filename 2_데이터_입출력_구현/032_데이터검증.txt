데이터 검증
- 원천 시스템의 데이터를 목적 시스템 데이터로 전환화는 과정이 정상적으로 수행되었는지 여부 확인 과정
- 검증 방법, 검증 단계에 따라 분류

검증 방법에 따른 분류
- 로그 검증 : 데이터 전환 과정에서 ETL 로그 검증
- 기본 항복 검증 : 별도로 요청된 검증 항목 검증
- 응용 프로그램 겅증 : 응용 프로그램 통한 데이터 전환의 정합성 검증
- 응용 데이터 검증 : 업무 규칙을 기준으로 데이터 전환의 정합성 검증
- 값 검증 : 합계 검증, 범위 검증, 값 검증

검증 단계에 따른 분류
- 추출 : 원천 시스템 데이터에 대한 정합성 확인 - 로그 검증
- 전환 : 매핑 정의서에서 정의된 내용 정확히 반영 되었는지, 오류 여부도 확인 - 로그 검증
- DB 적재 : SAM파일 적재 과정에서 오류나 데이터 누락 여부 등 확인 - 로그 검증
- DB 적재 후 : 적재 완료 후 정합성 확인 - 기본 항목 검증
- 전환 완료 후 : 추가 검증 과정으로 정합성 검증 - 응용 프로그램 검증, 응용 데이터 검증
