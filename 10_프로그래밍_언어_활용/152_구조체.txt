구조체

구조체의 개요
- 자료의 종류가 다른 변수의 모임 -> 문자, 숫자 등 혼용 가능
- Java는 클래스가 구조체임

구조체의 정의
- int나 char 같은 자료형을 하나 만드는 것
- 표기 방법 예시 : struct sawon { -> struct는 구조체 정의하는 예약어
		char name[10];	
		int pay;
		};

구조체 변수의 선언
- 구조체 변수를 사용하려면 먼저 정의한 구조체에 대한 변수를 선언해야 함
- 표기 방법 예시 : struct sawon ansan, *seoul -> sawon ansan은 sawon 자료형으로 변수를 선언하고 변수 이름이 ansan
				         -> sawon *seoul은 구조체의 포인터 변수

구조체 멤버의 지정
- 구조체 멤버 모든 요소들이 개별적 이름을 가지고 있어 구조체의 멤버를 지정할 때는 구조체 변수와 함께 멤버 이름을 지정해야 함
- '.' 의 의한 지정 : 구조체 일반 변수를 이용해 구조체 멤버를 지정할 때
-> ansan.name = "김한국"; ansan.pay = 4000000;

- '->'에 의한 지정 : 구조체 포인터 변수를 이용해 구조체 멤버를 지정할 때
-> seoul->name = "홍길동"; seoul->pay = 5000000;

- 구조체의 포인터 변수는 일반 포인터 변수처럼 *사용해 멤버 지정할 수 있음
-> (*seoul).name = "홍길동"; (*seoul).pay = 5000000;

- 한번에 지정하기
-> struct jsu st[2] = { {"김한국", 400000}, {"홍길동", 5000000} }; 하면 구조체 정의한 순서대로 데이터가 들어감.
-> 즉 st[0].name[0] ~st[0].name[9], st[0].pay 와 같이 배열에 저장되기 때문
