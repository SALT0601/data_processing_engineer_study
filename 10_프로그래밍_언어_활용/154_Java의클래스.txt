Java 클래스

Java 클래스의 개요
- 클래스는 객체 생성을 위한 필드(속성)와 메소드(함수)를 정의하는 설계도로 Java는 아무리 작은 프로그램이라도 클래스를 만들어서 사용해야 함
- 클래스를 만들어 사용하는 순서
-> 1. 클래스 이름을 정하고 객체 생성을 위한 필드와 메소드를 정의, 이때 사용하는 명령이 class
-> 2. 객체를 생성, new 명령 사용
-> 3. 생성된 객체들을 이용해 프로그램 코딩

- 생성자 : 객체 변수 생성에 사용되는 메소드, 객체 변수를 생성하면서 초기화를 수행한다.
- super : 상속한 부모 클래스를 가리킴 / super() -> 부모 크래스의 생성자 호출
- this : 현재 실행중인 메소드가 속한 클래스를 가리킴

- 재정의 :부모 클래스를 상속받은 자식 클래스에서 부모 클래스의 메소드를 다시 동일한 이름(인수까지 같아야함)으로 정의 
  -> 부모 클래스의 메소드가 아니라 자식 클래스의 메소 드가 호출됨

- 오버로딩 : 동일한 이름의 메소드로 인수만 달리해 여러 기능 정의
   -> ex. prn(), prn(int x), prn(double x) -> 다 다름

- 보통 메소드를 호출하려면 메소드가 포함된 클래스의 객체 변수를 선언한 후 [객체변수].[메소드]와 같이 호출
- 하지만 [클래스이름].[메소드]와 같은 방식으로 호출하는 경우에는 호출되는 메소드가 static으로 선언돼야 함
