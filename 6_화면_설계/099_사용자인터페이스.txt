사용자 인터페이스(UI)
- 사용자와 시스템 간 상호작용이 원활히 이뤄지도록 도와주는 장치나 소프트웨어
- 3가지 분야
-> 정보 제공과 전달을 위한 물리적 제어에 관한 분야
-> 콘텐츠의 상세적 표현과 전체적 구성에 관한 분야
-> 모든 사용자가 편리하고 간쳔하게 사용하도록 하는 기능에 관한 분야

사용자 인터페이스의 구분
- CLI(Command Line Interface) : 명령과 출력이 텍스트 형태로 이뤄짐
- GUI(Graphical User Interface) : 아이콘이나 메뉴를 마우스로 선택해 작업 수행하는 그래픽 환경의 인터페이스
- NUI(Natural User Interface) : 사용자의 말이나 행동으로 기기 조작

사용자 인터페이스의 기본 원칙
- 직관성 : 누구나 쉽게 이해하고 사용 가능
- 유효성 : 사용자의 목적 정확하고 완벽하게 달성
- 학습성 : 누구나 쉽게 배우고 익힘
- 유연성 : 사용자의 요구사항 최대한 수용하고 실수 최소화
