서비스 공격 유형

서비스 거부(DoS; Denial of Service) 공격
- 대량의 데이터를 한 곳의 서버에 집중적으로 전송함으로써 서버의 정상적인 기능 방해하는 것
- 주요 유형 : Ping of Death, SMURFING, SYN Flooding, TearDrop, LAND Attack, DDoS 공격

Ping of Death(죽음의 공격)
- Ping 명령 전송 시 패킷의 크기를 인터넷 프로토콜 허용 범위 이상으로 전송해 네트워크를 마비시키는 서비스 거부 공격 방법

SMURFING(스머핑)
- IP나 ICMP 특성을 악용해 ㅇ머청난 양의 데이터를 한 사이트에 집중적으로 보냄으로써 네트워크를 불능 상태로 만드는 공격 방법

SYN Flooding
- 가상의 클라이언트로 위장해 3-way-handshake 과정을 의도적으로 중단시킴으로싸 서버가 대기 상태에 놓여 정상적인 서비스를 수행하지 못하게 하는 공격 방법

TearDrop
- Fragment Offset 값을 변경시켜 수신측에서 패킷 재조립시 오류로 과부화 발생시킴으로써 시스템이 다운되도록 하는 공격 방법

LAND Attack(Loacal Area Network Denial Attack)
- 패킷을 전송할 때 송신 IP와 수신 IP 주소를 모두 공격 대상의 IP 주소로 하여 공격 대상에게 전송하는 것으로 자신에 대해 무한히 응답하게 하는 공격

DDoS(분산 서비스 거부) 공격
- 여러 곳에 분산된 공격 지점에서 한 곳의 서버에 대해 분산 서비스 공격을 수행
- 취약점에 있는 호스트에 분산 서비스 공격용 툴을 설치해 에이전트로 만든 후 DDoS 공격에 이용
- 분산 서비스 공격용 툴 : 에이전트의 역할을 수행하도록 설계된 프로그램으로 데몬이라 부름
-> Trin00 : 가장 초기 형태의 데몬, 주로 UDP Flooding 공격 수행
-> TFN : UDP Flooding 공격뿐만 아니라 TCP SYN Flood 공격, ICMP 응답 요청, 스머핑 공격 등 수행
-> TFN2K : TFN 확장판
-> Stacheldraht : 이전 툴들의 기능을 유지하면서 공격자, 마스터, 에이전트가 쉽게 노출되지 않도록 암호화된 통신 수행, 툴이 자동으로 업데이트 됨

네트워크 침해 공격 관련 용어
- 세션 하이재킹 : 상호 인증 과정을 거친 후 접속해 있는 서버와 서로 접속되어 클라이언트 사이의 세션 정보 가로챔(TCP 세션 하이재킹이 대표적) - 시퀀스 넘버 조작
- ARP 스푸핑 : ARP의 취약점 이용한 공격 기법
- 스미싱 : 문자 메시지를 이용해 사용자의 개인 신용 정보를 빼내는 수법
- 스피어 피싱 : 사회 공학의 한 기법으로 특정 대상을 선정한 후 그 대상에게 일반적인 이메일로 위장한 메일 지속적으로 발송해 메일의 본문링크나 첨부파일을 클릭하도록 유도해 탈취
- APT(지능형 지속 위협) : 조직적으로 특정 기업이나 조직 네트워크에 침투해 활동 거점 마련한뒤 때를 기다려 보안을 무력화시키고 정보 수집
- 무작위 대입 공격 : 암호키를 찾아내기 위해 적용 가능한 모든 값 대입해 공격
- 큐싱 : QR코드를 통해 악성 앱 다운 유도하거나 설치하도록 함
- SQL 삽입 : 전문 스캐너 프로그램 또는 봇넷 등 이용해 웹사이트를 무차별적으로 공격하는 과정에서 취약한 사이트 발견되면 데이터 베이스등 데이터 조작
- 크로스 사이트 스크립팅(XSS) : 스크립트의 취약점을 악용한 해킹 기법
- 스니핑 : 네트워크 중간에서 남의 패킷 정보를 도청하는 해킹 유형, 수동적 공격

정보 보안 침해 공격 관련 용어
- 좀비 PC : 악성코드에 감염되어 다른 프로그램이나 컴퓨터를 조종하도록 만들어진 컴퓨터로 주로 DDoS 공격에 이용
- C&C 서버 : 해커가 원격지에서 감염된 좀비 PC에 명령 내리고 악성 코드 제어하기 위한 용도로 사용하는 서버
- 봇넷 : 악성 프로그램에 감염되어 악의적인 의도로 사용될 수 있는 다수의 컴퓨터들이 네트워크로 연결된 형태
- 웜 : 네트워크를 통해 연속적으로 자신을 복제해 시스템 부화 높여 다운시키는 바이러스의 일종, DDoS가 웜 공격의 한 형태
- 제로 데이 공격 : 보안 취약점이 발견되었을 때 발견된 취약점의 존재 차제가 널리 공표되기 전 해당 취약점 통해 이루어지는 보안 공격, 공격의 신속성 의미
- 키로거 공격 : 컴퓨터 사용자의 키보드 움직임 탐지해 개인 정보 빼가는 해킹
- 랜섬웨어 : 인터넷 사용자의 컴퓨터에 잠입해 내부 문서나 파일등 암호화해 사용자가 열지 못하게 하여 돈 요구
- 백도어 : 시스템 설계자가 서비스 기술자나 유지 보수 프로그램 작성자의 액세스 편의를 위해 시스템 보안을 제거하여 만들어놓은 비밀 통로, 범죄에 악용되기도 함
- 트로이 목마 : 정상적인 기능을 하는 프로그램으로 위장해 숨어있다가 해당 프로그램 동작할 때 활성화해 부작용 일으킴
