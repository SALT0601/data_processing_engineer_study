유스케이스(Use Case) 다이어그램

기능 모델링
- 사용자가 요구한 개발될 시스템이 갖춰야 할 기능을 정리, 사용자와 함께 내용을 공유하기 위해 그림으로 현
- 전반적인 형태를 기능에 초점을 맞춰 표현
- 기능 모델링의 종류
=> 유스케이스 다이어그램, 액티비티 다이어그램

유스케이스(Use Case) 다이어그램
- 개발될 시스템을 이용해 수행할 수 있는 기능을 사용자 관점에서 표현
- 외부 요소와 시스템 간 상호 작용 확인 가능
- 사용자 요구사항 분석을 위한 도구
- 시스템 범위 파악 가능

유스케이스 구성 요소
- 시스템/ 시스템 범위 - 사각형 - 시스템 내부의 유스케이스들 묶어 범위 표현
- 액터 - 시스템과 상호작용하는 모든 외부 요소, 주로 사람이나 외부 시스템
=> 주액터 : 사람모양 - 시스템을 사용하면 이득을 얻는 대상, 주로 사람
     부액터 : 《Actor》 쓰고 밑에 사각형 - 주액터의 목적 달성을 위해 서비스를 제공하는 외부 시스템, 조직이나 기관 등
- 유스케이스 - 타원형 - 사용자가 보는 관점에서 서비스나 기능 
-  관계 - 액터와 유스케이스, 유스케이스와 유스케이스 
=> 포함 -  《Actor》 쓰고 밑에 포함되는 유스케이스쪽으로 점선 화살표 
     확장 - 《Actor》 쓰고 원래의 유스케이스쪽으로 점선 화살표
     일반화 - 속이 빈 화살표
